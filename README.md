# remote-execution-service
This is a repo containing Sourcecode of remote execution through sms. It's mainly focused on termux, python (django 6.0 or later.) Execution happens via sms auth.


Usage : 
Start the service by using command, 
python (standard or 3 ) manage.py (path where this file is located) start_service (main argument).

#### Hence becomes 
``` python manage.py start_service ```

## Warning, this can be used at own risk, and must have prior knowledge of execution.
